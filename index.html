Firebase Setup Required

Follow these 5 steps once, then the game is ready to share with your team

1

Create a Firebase project

Go to console.firebase.google.com → Add project → name it (e.g. office-spin) → click through → Create project.

2

Enable Google Sign-In

Firebase console → Authentication → Sign-in method → click Google → toggle Enable → add your support email → Save.

3

Create Firestore Database

Firebase console → Firestore Database → Create database → Production mode → pick region (e.g. asia-south1) → Done.

Then go to the Rules tab and replace the contents with:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /spins/{docId} {
      allow create: if request.auth != null;
      allow read, delete: if request.auth != null
        && request.auth.token.email in ['you@company.com'];
    }
  }
}

Replace you@company.com with your admin email, then click Publish.

4

Register a Web App & get config

Firebase console → Project Settings (gear icon) → scroll to Your apps → click </> Web → enter an app nickname → Register app. Copy the firebaseConfig object shown.

Also in Project Settings → Authorized domains → add the domain where this file will be hosted (for local testing, localhost is already allowed).

5

Paste your config into this file

Open this HTML file in any text editor. Find the firebaseConfig block inside the <script type="module"> tag at the bottom and replace all YOUR_* values with your real keys. Also update the ADMIN_EMAILS array with your email. Save → reload.
